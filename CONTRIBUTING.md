# Contributing to SmartStack

First off, thank you for considering contributing to SmartStack! It's people like you that make the project great.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub issue tracker.
- Describe the bug and provide steps to reproduce it.

### Suggesting Enhancements
- If you have an idea for a new feature, feel free to open an issue for discussion.

### Pull Requests
1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Commit your changes with descriptive messages.
4. Push to your branch and open a Pull Request.

## Coding Standards
- **CSS:** Use Vanilla CSS with variables defined in `Global.css`. Keep the glassmorphic aesthetic consistent.
- **Python:** Follow PEP 8 standards. Use environment variables for sensitive data.
- **React:** Use functional components and hooks. Keep components clean and modular.

## Development Setup
Refer to the [README.md](./README.md) for detailed setup instructions.
